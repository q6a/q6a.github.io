Kali ini, kami akan membahas cara agar kalian bisa mengimpor dan mengedit file SRT ke Adobe Premiere Pro.

Sebelum mulai, yuk kita cari tahu apa sih file SRT…

Apa Itu SRT?

SRT adalah file teks biasa yang berisi informasi penting mengenai subtitle, termasuk kode waktu awal dan akhir teks kalian untuk memastikan subtitle kalian cocok dengan audio kalian, dan nomor urut subtitle.

Dengan memiliki SRT, video kalian akan memiliki subtitle atau teks di layar. Memiliki subtitle atau teks pada video bakalan sangat membantu banyak orang, loh, terutama mereka yang memiliki gangguan pendengaran.

Bagaimana Cara Membuat File SRT?

Secara garis besar, kalian bisa bisa membuat file SRT dengan menggunakan editor teks apapun atau dengan menggunakan software SRT.

Menggunakan Editor Text Untuk Membuat File SRT

Untuk membuat file SRT di editor text, gunakan langkah-langkah berikut:

  1. Buka text editor apa saja, misalnya TextEdit atau Notepad.
  2. Urutkan garis caption dengan nomor yang dimulai dari angka 1.
  3. Tekan enter dan tempatkan cap waktu. Format dari cap waktu yang akan kalian tempatkan adalah ‘jam:menit:detik,milidetik –> jam:menit:detik,milidetik’.
  4. Tekan enter lagi dan mulailah menulis caption kalian.

Pastikan kalian tidak mengacaukan format cap waktu.

Menyimpan FIle SRT Dari TextEdit Di Mac

Buka aplikasi TextEdit di Mac kalian.

  1. Urutkan garis caption dengan nomor yang dimulai dari angka 1.

  2. Tekan enter dan tempatkan cap waktu. Format dari cap waktu yang akan kalian tempatkan adalah ‘jam:menit:detik,milidetik –> jam:menit:detik,milidetik’.

  3. Tekan enter lagi dan mulailah menulis caption kalian.

  4. Klik Format > ‘Make Plain Text’.

    Pastikan kalian sudah mengatur 'make plain text' di TextEdit untuk membuat file SRT

  5. Klik File > save.

  6. Di kolom Save As, ketik nama file kalian. TextEdit akan secara otomatis membuat extension file kalian menjadi .txt jadi kalian harus menggantinya menjadi .srt.

    Menyimpan file SRT di TextEdit

  7. Hapus centang tulisan ‘If no extension is provided, use .txt’ dan klik save.

Menggunakan VideoTranslator Untuk Membuat File SRT

Kalau kalian tidak mau membuat file SRT secara manual, kalian bisa menggunakan VideoTranslator. VideoTranslator menggunakan kecerdasan buatan untuk mentranskripsikan video kalian. Dengan menggunakan VideoTranslator, kalian bisa menghemat banyak waktu.

Berikut merupakan langkah-langkah yang harus kalian lakukan untuk membuat file SRT dari VideoTranslator:

  1. Kunjungi halaman www.videotranslator.ai

  2. Kalau kalian sudah memiliki akun, masuk ke akun kalian. Kalau kalian belum memiliki akun, kalian. bisa membuat akun baru atau melakukan percobaan gratis.

  3. Unggah video kalian melalui tombol quicktranscribe.

  4. Pilih bahasa dan dialek dari video kalian dan pencet tombol accept untuk melakukan transkripsi.

    Menggunakan VideoTranslator untuk membuat file SRT

  5. Setelah proses transkripsi selesai, cek kembali hasil transkripsi dan benarkan tata bahasa seperti huruf kapital dan tanda titik.

  6. Unduh file srt.

Cara Mengimpor File SRT Ke Adobe Premiere Pro

Nah, setelah kalian tau gimana sih cara membuat file SRT, kalian bisa mengimpor file kalian ke Adobe Premiere Pro.

Berikut adalah cara untuk mengimpor file SRT di Adobe Premiere Pro:

  • Buka proyek kalian di Adobe Premiere Pro. Adobe Premiere Pro yang kami gunakan adalah versi 2020.
  • Impor subtitle dengan mengklik File > Impor dan temukan file subtitle kalian kemudian klik impor.

Atau, kalian bisa menyeret file subtitle ke timeline.

Cara Mengedit File SRT Ke Adobe Premiere Pro

Ada dua cara untuk mengedit subtitle di Adobe Premiere Pro.

Cara Pertama Untuk Mengedit Subtitle Di Adobe Premiere Pro

  1. Pergi ke Window > Caption

    Klik tombol caption untuk mengedit subtitle kalian di Adobe Premiere Pro

  2. Klik import settings. Disini, kalian bisa melakukan pengaturan subtitle seperti font style, font size, dan lokasi subtitle secara keseluruhan.

    Klik tombol import untuk mengedit subtitle kalian di Adobe Premiere Pro

  3. Dengan melakukan metode ini, seluruh pengaturan subtitle kalian akan seragam.

    Settingan caption di Adobe Premiere Pro

Cara Kedua Untuk Mengedit Subtitle Di Adobe Premiere Pro

  1. Pastikan bahwa file SRT kalian sudah ada di timeline.
  2. Klik dua kali di file subtitle kalian.
  3. Halaman caption akan dibuka. Disini, kalian bisa mengatur subtitle kalian satu persatu.

Metode ini bagus jika kaliaj hanya ingin mengedit beberapa bagian dari subtitle.

Mengedit subtitle di Adobe Premiere Pro secara manual

Apa yang Harus Saya Lakukan Jika Saya Tidak Dapat Mengimpor File SRT Ke Adobe Premiere Pro?

Jika kalian tidak bisa mengunggah file SRT kalian ke Adobe Premiere Pro, kalian bisan mencoba cara berikut ini:

  1. Klik ikon kunci inggris di timeline.

    Cara untuk mengimport file SRT ke Adobe Premiere Pro

  2. Pergi ke Closed Captions Display dan pastikan bahwa kalian sudah mengaktifkan fungsi ini dengan mengklik ‘enable’.

    Pastikan fungsi 'enable closed captions' di Adobe Premiere Pro sudah diaktifkan untuk menambahkan subtitle

Mengekspor Video Dengan Subtitle Dalam Adobe Premiere Pro

Untuk mengekspor video dengan subtitle di Adobe Premiere Pro, kalian dapat melakukan hal berikut:

  • Klik File > Export. Atau, kalian bisa memencet tombol command + M (untuk Mac).
  • Pilih pengaturan export dan preset.
  • Klik export.

Video kalian akan secara otomatis memiliki subtitle yang sudah tertanam.
